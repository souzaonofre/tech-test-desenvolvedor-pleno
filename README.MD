# Teste Técnico – Desenvolvedor Ruby on Rails (Pleno)

## Objetivos

- O candidato deverá desenvolver uma aplicação Ruby on Rails para processar arquivos .eml (e-mails) e extrair informações estruturadas,
	salvando os resultados em banco de dados.​

- O projeto deve contemplar arquitetura limpa, background jobs, logs persistentes e interface web.
